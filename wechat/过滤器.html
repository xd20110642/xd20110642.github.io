    # 小程序使用过滤器

    • 在小程序中我们没有办法直接使用类似vue react框架中的定义的过滤器,我们必须变通的使用。

    • 微信 为我们提供了一种叫wxs的文件 用这个文件就可以变通的使用过滤器了。

    • 首先我们需要创建一个wxs文件:

        每一个 .wxs 文件和 <wxs> 标签都是一个单独的模块。
  每个模块都有自己独立的作用域。即在一个模块里面定义的变量与函数，默认为私有的，对其他模块不可见。
  一个模块要想对外暴露其内部的私有变量与函数，只能通过 module.exports 实现。

  //我们定义的处理数据的方法
  var wetherFilters = function (value) {
  if (!value) return value;
  var _lowTem = value.indexOf('-') > -1;
  var _value = value.split(" ")[1];//切割为数组
  _value = _lowTem ? '-' + _value : _value
  return _value
  }
  然后再导出:
  module.exports = {
      wetherFilters: wetherFilters,
  }

    • 使用 在我们的wxml文件中引入这个wxs文件

        <wsx module="自定义的引用名字" src="wxs的文件路径">
  <wsx module="data" src="wxs的文件路径">

    • 使用

        <view>
     自定义的引用名.需要用到的方法(传递数据)
  </view>


  这样就完成了在微信中的自定义过滤器的使用

    • 备注:在wsx文件中 目前只能使用ES5语法 暂不支持ES6语法
